Download Verity & Companion Apps

Desktop (Verity)

Install the desktop app for Windows, macOS, or Linux.

  • Windows installer (.exe) — 64-bit
  • macOS installer (.dmg)
  • Linux (.AppImage)

If you need offline installers or other architectures, contact team@veritysystems.app.

Browser Extension — Verity Companion App

Install from the Chrome Web Store when published (link will appear here after publishing).

Dev install: open chrome://extensions, enable Developer Mode → "Load unpacked" → select browser-extension/chrome/

Publish status: Not Published

Web / PWA — Verity Companion App

Your PWA is available at: https://www.veritysystems.app/

To install: open site in Chrome/Edge and use the browser install prompt or select "Install" from browser menu.

Mobile (Native)

Native builds are available via the App Store / Play Store after publishing.

If you need an ad-hoc or enterprise build, contact team@veritysystems.app.

Developer & Release Tools

  • scripts/publish_extension.sh — Package and upload Chrome extension (requires API credentials)
  • scripts/publish_desktop_release.sh — Create GitHub release and upload assets (uses gh CLI)
  • scripts/purge_vercel_cache.sh — Redeploy / force refresh Vercel cache
