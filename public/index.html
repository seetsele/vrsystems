Developer API

Simple, Powerful REST API

Integrate fact-checking into your applications with our intuitive API. Choose your preferred language and get started in minutes.

bash cURL Command 
curl -X POST https://veritysystems-production.up.railway.app/v3/verify \
    -H "X-API-Key: YOUR_API_KEY" \\
  -H "Content-Type: application/json" \\
  -d '{
    "claim": "The Earth is approximately 4.5 billion years old",
    "providers": ["anthropic", "groq", "wikipedia"],
  }'
python Python Script 
import requests

response = requests.post(
    "https://veritysystems-production.up.railway.app/v3/verify",
    headers={"X-API-Key": "YOUR_API_KEY"},
    json={
        "claim": "The Earth is approximately 4.5 billion years old",
        "providers": ["anthropic", "groq", "wikipedia"]
    }
)

result = response.json()
print(f"Status: {result['status']} (confidence {result['confidence_score']})")
javascript JavaScript / Fetch API 
const response = await fetch('https://veritysystems-production.up.railway.app/v3/verify', {
  method: 'POST',
  headers: {
        'X-API-Key': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    claim: 'The Earth is approximately 4.5 billion years old',
        providers: ['anthropic', 'groq', 'wikipedia']
  })
});

const result = await response.json();
console.log(`Status: ${result.status} (confidence ${result.confidence_score})`);
javascript Node.js with Axios 
const axios = require('axios');

async function verifyClaim() {
    const response = await axios.post(
        'https://veritysystems-production.up.railway.app/v3/verify',
        {
            claim: 'The Earth is approximately 4.5 billion years old',
            providers: ['anthropic', 'groq', 'wikipedia']
        },
        {
            headers: {
                'X-API-Key': 'YOUR_API_KEY'
            }
        }
    );
  
    console.log(`Status: ${response.data.status} (confidence ${response.data.confidence_score})`);
}

verifyClaim();
