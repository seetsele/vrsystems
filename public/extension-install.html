Install the Verity Chrome Extension

The Chrome extension cannot be installed silently from a web page for security reasons. Follow these steps to install it locally or use enterprise deployment for managed installs.

Local developer install (quick)

  1. Open the Extensions page in Chrome:
  2. Enable Developer Mode (top-right).
  3. Click Load unpacked, then select the extension source directory: browser-extension/chrome.
  4. The extension will load and you can pin it to the toolbar.

Enterprise / managed install (recommended for teams)

Use Chrome Browser Cloud Management or an enterprise policy to deploy the extension silently across machines. See your browser vendor documentation for "Extension management" or contact your IT admin for help.
