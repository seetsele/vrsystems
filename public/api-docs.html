Quick Start

Get started with the Verity API in minutes. Make your first fact-check request with just a few lines of code.

1. Get Your API Key

Sign up for a free account and generate your API key from the API Keys dashboard.

2. Make Your First Request

# Python Example
import requests

response = requests.post(
    "https://veritysystems-production.up.railway.app/v3/verify",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "claim": "The Earth is approximately 4.5 billion years old",
        "quick": False  # False = Full 9-Point (default)
    }
)

result = response.json()
print(f"Verdict: {result['verdict']}")
print(f"Verification: {result['verification']['level']}")
// JavaScript Example - Full 9-Point Verification (default)
const response = await fetch('https://veritysystems-production.up.railway.app/v3/verify', {
    method: 'POST',
    headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        claim: 'The Earth is approximately 4.5 billion years old'
        // quick: true  // Uncomment for fast preliminary check
    })
});

const result = await response.json();
console.log(`Verdict: ${result.verdict} (${result.verification.level})`);

Security

Security is our top priority. Here's how we protect your data and API access.

TLS 1.3 Encryption SOC 2 Type II GDPR Compliant Rate Limiting API Key Rotation Request Signing

Best Practices

  • Never expose API keys in client-side code. Use environment variables and server-side proxies.
  • Rotate keys regularly. Generate new keys every 90 days and immediately after any suspected compromise.
  • Use IP allowlisting. Restrict API access to known IP addresses for production keys.
  • Monitor usage. Set up alerts for unusual API activity in your dashboard.
