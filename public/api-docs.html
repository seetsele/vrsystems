API Documentation

Integrate Verity's 9-Point Triple Verification into your applications. Verify claims, detect misinformation, and analyze media programmatically.

Authentication

All API requests require an API key passed in the Authorization header:

HTTP Header
Authorization: Bearer YOUR_API_KEY

Get your API key from the API Keys dashboard.

Quick Start

Verify your first claim in seconds:

cURL
curl -X POST https://veritysystems-production.up.railway.app/v3/verify \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"claim": "The Earth is flat"}'

Verify Endpoint

POST/v3/verify

Verify a single claim using 9-Point Triple Verification.

Request Body

ParameterTypeDescription
claim*stringThe claim or statement to verify
optionsobjectOptional verification settings
options.source_checkbooleanEnable source verification (default: true)
options.bias_checkbooleanEnable bias analysis (default: false)

Response

JSON
{
  "verdict": "False",
  "confidence": 95,
  "summary": "The claim that Earth is flat is false...",
  "sources": [
    { "url": "https://nasa.gov/...", "title": "NASA Earth Data" }
  ],
  "models_used": 12,
  "processing_time": 2.3
}
